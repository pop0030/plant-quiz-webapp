# Plant quiz webapp

[Demo](https://pop0030.github.io/plant-quiz-webapp/)

> This project is build by `@vue-cli`

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
